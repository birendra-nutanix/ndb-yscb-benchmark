How to run the benchmarks:
# Extract the ZIP file
unzip 

# Run all benchmarks in parallel
chmod +x run_all_benchmarks.sh
./run_all_benchmarks.sh

# Or run individual database benchmarks
chmod +x ycsb_<database_name>.sh
./ycsb_<database_name>.sh

Prerequisites: YCSB must be installed at /ycsb/bin/ycsb. Check the README.md in the package for detailed instructions.
